Contoh Property CSS target Selector P: 

        
        p {
            color : yellow;
            margin: 20px;
            font-size: 24px
        }
Digunakan untuk menandai potongan kode bahasa program dalam satu baris. Ini memberikan makna semantik dan visual khusus untuk teks kode, seperti font monospaced, tapi kaga menjaga format spasi, indentasi atau baris baru. 
Digunakan untuk menampilkan blok teks yang mempertahkan format aslinya (termasuk spasi, indentasi dan baris baru). Biasanya digunakan untuk menampilkan kode pemrograman atau teks dengan format yang lebih spesifik.
